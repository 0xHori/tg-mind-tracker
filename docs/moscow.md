# MoSCoW (Черновик)

## Must (обязательно)
- Мини-ритуал: проговаривание фраз
- Мини-шаги по задаче: реально сделанное действие
- Вечерний журнал: фиксация эмоций

## Should (желательно)
- Завершение хотя бы одной задачи полностью
- Дополнительные фразы поддержки

## Could (опционально)
- Автоматические уведомления / кнопки
- Графики прогресса
- Случайные мотивационные советы

## Won’t (не на MVP)
- Сложные AI-анализы
- Полноценные визуализации, отчёты
